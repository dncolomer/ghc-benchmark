Genuine Human Cognition

Measuring how closely AI Chain of Thought matches real human thinking patterns

What's the Context?

Modern AI is overwhelmingly trained on finalized, edited, high-quality text—never on the live, uncertain, error-filled process of human discovery. This creates a fundamental gap: AI can sound polished, but it doesn't capture the raw, messy authenticity of genuine human cognition.

GHC (Genuine Human Cognition) is a benchmark that measures how closely AI's Chain of Thought (CoT) matches real human thinking patterns. We use recordings of actual humans working through Grover's algorithm to establish the ground truth for authentic cognition.

The Problem

When AI responds to user queries, it produces polished, structured outputs that rarely reflect how humans actually think. The result: AI assistants that feel robotic, never truly connecting with the messy, non-linear way humans process ideas.

Our Solution

By systematically collecting and analyzing authentic human cognition—from real people thinking out loud while solving problems—we now have the missing signal needed to measure whether AI truly understands human thought.

Three Benchmarks

The GHC benchmark uses three complementary metrics to evaluate how authentically AI thinks.

01 / AUTHENTICITY

Similarity Score

Uses sentence embeddings to measure how similar AI-generated thinking is to real human transcripts. Higher scores = closer match to human patterns.

0-100% via embedding similarity

02 / LINEARITY

Linearity Index

Measures how sequentially linear the thinking flows. Human thinking is fragmented with jumps and tangents; AI tends to be overly structured.

0-100 (higher = more linear/robotic)

03 / CLUSTERS

Cluster Score

Counts consecutive linear transitions. Humans have many short clusters (fragmented thinking). LLMs have long sustained chains (>5 = LLM-like).

0-100 (clusters >5 = robotic)

How to Run

Installation

git clone https://github.com/dncolomer/ghc-benchmark
cd ghc-benchmark
pip install -r requirements.txt

Running the Benchmark

# Run full pipeline (generate + evaluate + report)
python benchmark.py run --stage all

# Run individual stages
python benchmark.py run --stage generate   # Generate LLM samples
python benchmark.py run --stage evaluate    # Run benchmarks
python benchmark.py run --stage report       # Generate reports

# With specific models
python benchmark.py run --stage all --models "openai/gpt-4o-mini" "google/gemini-2.0-flash-001"

Output

Results are saved to results/scores.json and visualizations to results/charts/. A markdown report is generated in results/reports/report.md.
